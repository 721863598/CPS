# CPS v1.0.0 — 首个内测版发布 🎉

> **⚠️ 仅适配 Windows 系统**

---

## 📖 简介

CPS 是一款轻量级的**专注计时器**，帮助你记录每天的专注时长，并通过数据可视化的方式回溯你的时间投入。软件以系统托盘常驻运行，随开随用，不打扰你的工作流。

**作者：王梓赫** | 当前为**内测版本**，功能仍在持续完善中，欢迎试用并提出宝贵意见。

---

## 🔐 你的数据，只属于你

> ### 🏠 100% 本地离线存储 · 零云端上传
>
> CPS **不会、也不能**主动联网上传任何数据。
> 没有后台服务器、没有云同步、没有遥测上报——
> 你的计时记录**从始至终只存在于你自己的电脑上**，
> 数据的**所有权和掌控权完全属于你**。
> 断网环境完全正常使用，隐私安全无需担忧。

---

### 📁 数据中枢：`records` 文件夹

安装目录下的 `records` 文件夹是 CPS 的**唯一数据中枢**，所有计时记录以 **JSON 明文**存储在 `records.json` 中——任何文本编辑器均可直接打开查看。

| ⚠️ 重要声明 | 详细说明 |
|:---:|---|
| 🔓 **明文存储** | 数据以 JSON 明文格式保存，任何人都可以用记事本打开查看。请注意保管，**不要将 `records` 文件夹分享给他人** |
| 💻 **本地唯一** | 数据仅保存在本地安装目录，**不存在任何云端副本**，删了就没了 |
| ❌ **无自动备份** | CPS **不提供数据备份功能**，请自行定期复制 `records` 文件夹到 U 盘、移动硬盘等安全位置 |
| 🚫 **误删不负责** | 若 `records.json` 被手动删除、覆盖或损坏，**数据将永久丢失，软件无法恢复** |
| ✅ **卸载会保留** | 通过系统"卸载程序"正常卸载时，安装包会**自动保留** `records` 文件夹，下次重装可恢复；但手动删除整个安装目录会一并清除数据 |

> 💡 **养成好习惯**：每隔一段时间把 `records` 文件夹复制一份到其他地方存着，几秒钟的操作就能避免数据丢失的风险。

## ✨ 主要功能

### ⏱ 三圈时钟计时

- **外环（24h）**：以 24 小时为满圈，展示当前计时的小时数
- **中环（1h）**：以 60 分钟为满圈，展示当前计时的分钟数
- **内环（1min）**：以 60 秒为满圈，展示当前计时的秒数
- 三圈同步运转，时间流逝一目了然

### 🏷 任务标签

- 可为每次计时任务命名（例如"写论文"、"背单词"）
- 标签会随计时记录一起保存，方便后续回顾

### 📊 数据汇总

- **日 / 周 / 月 / 年**四个维度查看专注数据
- 今日专注总时长、任务数量、最长任务一目了然
- 支持时段分布柱状图，发现你的高效时间段
- 标签汇总，看清时间都花在哪里

### 🔥 计时热力图

- 类似 GitHub 贡献图的年度热力图
- 月份排行、周活跃 Top 12、日活跃 Top 12
- 直观展示你的专注趋势

### 🪟 迷你模式

- 双击表盘可切换至 **104×104 紧凑圆形窗口**
- 迷你窗口始终置顶，支持拖拽移动
- 适合需要持续关注时间又不占用屏幕空间的场景

### 📋 系统托盘常驻

- 关闭窗口自动缩至托盘，不退出程序
- 右键托盘图标可显示窗口或退出
- 计时进行中退出时会弹窗确认，防止误关丢失记录

### 🔒 单实例运行

- 同时只能运行一个 CPS，重复点击不会创建多个窗口

## 📥 安装与更新

1. 下载 `CPS Setup 1.0.0.exe`
2. 双击运行，**简介页**点击"下一步"
3. 选择安装目录（会自动追加 `\CPS` 文件夹）
4. 等待安装完成即可

> 如果之前已安装过旧版本，再次运行安装包会**在原位置自动更新**，无需手动卸载。
